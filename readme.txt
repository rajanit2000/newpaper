== Description ==
New Paper is simple responsive SEO friendly theme. 

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in New Paper in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away..
5. Navigate to Appearance > Customize in your admin panel and customize it.
